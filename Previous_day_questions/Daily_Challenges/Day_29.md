## 1. Valid Parentheses (LeetCode #20)
A problem that teaches stack operations and string validation techniques by checking whether a string of brackets is properly matched. Given a string s containing just the characters '(', ')', '{', '}', '[' , ']', determine if the input string is valid. An input string is valid if every open bracket is closed by the same type of bracket and brackets are closed in the correct order.

This problem is commonly asked in interviews and helps build concepts like:
- Stack operations
- String validation
- Bracket matching

which are important for efficient data structure based problems.

**Your task:** Determine whether the given string of brackets is valid using an optimized approach.

### Examples

**Input:**
```
s = "()[]{}"
```
**Output:**
```
true
```

---

**Input:**
```
s = "(]"
```
**Output:**
```
false
```

---

**Input:**
```
s = "([])"
```
**Output:**
```
true
```

## 2. Reverse a String Using Stack
A problem that teaches stack operations and string manipulation techniques by reversing a string using a stack data structure. Given a string s, reverse the string using stack operations and return the reversed string.

This problem is commonly asked in interviews and helps build concepts like:
- Stack operations
- String traversal
- LIFO processing

which are important for efficient data structure based problems.

**Your task:** Reverse the given string using a stack and an optimized approach.

### Examples

**Input:**
```
s = hello
```
**Output:**
```
olleh
```

---

**Input:**
```
s = stack
```
**Output:**
```
kcats
```

---

**Input:**
```
s = a
```
**Output:**
```
a
```
