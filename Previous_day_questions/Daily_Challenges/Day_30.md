## 1. Remove All Adjacent Duplicates In String (LeetCode #1047)
A problem that teaches stack operations and string reduction techniques by repeatedly removing adjacent duplicate characters. Given a string s consisting of lowercase English letters, perform the following operation until it can no longer be applied: choose two adjacent and equal letters in the string and remove them. After all such removals, return the resulting string. It can be shown that the answer is unique.

This problem is commonly asked in interviews and helps build concepts like:
- Stack operations
- String reduction

which are important for efficient string simplification problems.

**Your task:** Remove all adjacent duplicate characters from the given string using an optimized approach.

### Examples

**Input:**
```
s = "abbaca"
```
**Output:**
```
"ca"
```

---

**Input:**
```
s = "azxxzy"
```
**Output:**
```
"ay"
```

## 2. Insert Element at Bottom of Stack
A problem that teaches stack manipulation by inserting an element at the bottom of a stack. Given a stack and an integer value x, insert the element at the bottom of the stack while maintaining the order of the remaining elements.

This problem is commonly asked in interviews and helps build concepts like:
- Stack operations
- Element rearrangement

which are important for efficient stack based problems.

**Your task:** Insert the given element at the bottom of the stack using an optimized approach.

### Examples

**Input:**
```
Stack = [1,2,3,4], x = 5
```
**Output:**
```
[5,1,2,3,4]
```

---

**Input:**
```
Stack = [7,8], x = 1
```
**Output:**
```
[1,7,8]
```
