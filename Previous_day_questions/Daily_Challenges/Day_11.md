## 1. Isomorphic Strings (LeetCode #205)
A problem that focuses on comparing two strings to determine whether they follow the same character pattern. Given two strings s and t, determine whether they are isomorphic. Two strings are isomorphic if every character in one string can be consistently replaced to obtain the other string while preserving the order of characters.

This problem helps build concepts like:
- String traversal
- Character comparison
- Pattern recognition
- Basic string manipulation

which are important for solving a variety of string-based problems.

**Your task:** Determine whether the given strings are isomorphic using an optimized approach.

### Examples

**Input:**
```
s = "egg", t = "add"
```
**Output:**
```
true
```

---

**Input:**
```
s = "foo", t = "bar"
```
**Output:**
```
false
```

---

**Input:**
```
s = "paper", t = "title"
```
**Output:**
```
true
```

## 2. String Processing Task
A problem that focuses on modifying a string by applying a sequence of simple character operations. Given a string containing uppercase and lowercase English letters: Remove all vowels. Add a '.' before every consonant. Convert uppercase consonants to lowercase. Consider A, E, I, O, U (and their lowercase forms) as vowels.

This problem helps build concepts like:
- String traversal
- Character filtering
- Case conversion
- Basic string manipulation

These concepts are useful for solving a variety of string-processing problems.

**Your task:** Transform the given string by following the specified rules.

### Examples

**Input:**
```
Hello
```
**Output:**
```
.h.l.l
```

---

**Input:**
```
Programming
```
**Output:**
```
.p.r.g.r.m.m.n.g
```

---

**Input:**
```
aBAcAba
```
**Output:**
```
.b.c.b
```
